## Getting Started

To get started with this project, you can clone the repository and install the dependencies:

```sh
git clone https://github.com/EnzoVieira/instagram-clone.git
cd instagram-clone

yarn
## or
npm install
```

You can then start the development server:

```sh
npm run start
```
